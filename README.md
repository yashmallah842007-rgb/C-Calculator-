# C++ Simple Calculator

This is a simple calculator project written in C++.

## Features
- Addition  
- Subtraction  
- Multiplication  
- Division (with division-by-zero check)

## How to run

1. Make sure you have g++ installed.
2. Compile the program:

```bash
g++ calculator.cpp -o calculator
```

3. Run it:

```bash
./calculator
```

## Code File
The main program logic is inside:

```
calculator.cpp
```

## Description
The calculator allows you to:
1. Add two numbers  
2. Subtract two numbers  
3. Multiply two numbers  
4. Divide two numbers  
5. Exit the program  

You select an option and enter two integers. The program prints the result.
